## Most accomplished

Below we look at which 5 characters from Star Wars are the most
accomplished pilot (flew the fewest number of starships):

| Character      | Number of Starships piloted |
| :------------- | --------------------------: |
| R2-D2          |                           7 |
| C-3PO          |                           6 |
| Obi-Wan Kenobi |                           6 |
| Chewbacca      |                           5 |
| Leia Organa    |                           5 |
| Luke Skywalker |                           5 |
| Palpatine      |                           5 |
| Yoda           |                           5 |
| Darth Vader    |                           4 |
| Han Solo       |                           4 |
